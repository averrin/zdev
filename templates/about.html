Нафига это все?

Ну мало ли какая мелкая штука понадобится, а тут бац и почти фреймворк. Лафа, согласитесь.

Как это все?

Если же вам понадобилось что-то сюда добавить, то вот основные ссылочки:

  • Twitter Bootstrap CSS
  • Django Python
  • LESS CSS
  • CoffeeScript JS

Что касается самого проекта, то структура обычная для Django-проектов версии 1.4:

  • Cтранички хранятся в templates/*.html
  • Статика в папке static
  • Код разбит на приложения в соответствии с логикой
Ознакомится с основами можно тут: Django Documentation

Как, и это все?

Ну вообще, нет. Есть желание сделать ajax-навигацию на Backbone + CoffeeScript. У меня даже есть исходники, просто пока не приделывал.
А еще с Bootstrap'ом хорошо идет LESS.
Так что, если хочется поковырять няшки — здесь самое место, более того, уже даже все готово. Просто начинайте писать сразу в LESS или Coffee, а нужные библиотеки уже подцеплены.
